Fonctionnalités clés :

1. Recherche de films : Implémentation d'une barre de recherche dynamique permettant de trouver des films par titre ou genre, avec tri par popularité (Top/Flop).

2. Gestion des favoris : Ajout et suppression de films dans une liste de coups de cœur stockée localement.

3. Affichage des films hebdomadaires : Présentation des films en salle chaque semaine avec la possibilité de visionner les bandes-annonces directement sur l'application.

4. Carrousel de bandes-annonces : Création d'un carrousel en construction pour afficher les bandes-annonces des films à venir.

5. Optimisation des performances : Utilisation de useMemo pour mémoriser les mappings de genres afin d'optimiser les performances de l'application.

Technologies :

-React.js, React Router
-Axios pour les requêtes HTTP
-Moment.js pour la gestion des dates
-Local Storage pour la gestion des favoris
-SCSS pour le style
